## 📌 About The Project

EngineersClassroom.in is a web-based platform developed to help students learn engineering tools like BIM, MEP, AutoCAD and other technical skills in a simple and structured way.

This project focuses on making technical learning more accessible and beginner-friendly.

---

## ✨ Features

* 📚 Educational platform for engineering students
* 🎯 Clean and simple UI
* 📱 Responsive design (mobile + desktop)
* ⚡ Fast and optimized performance
* 🌐 Live deployed website
